#### 二、固件特色
* TUF-BE3600 最新官方固件有降级检测机制，建议刷入固件前，利用 SSH 关闭固件降级检测;
* **SSH 登录路由器后台后输入命令：**
```
nvram set DOWNGRADE_CHECK_PASS=1
```
