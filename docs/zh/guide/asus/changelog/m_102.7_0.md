**更新日志：**

* 合并梅林上游102.7_0源代码
* 梅林固件有的功能，梅林改版固件都有；
* 梅林固件不能使用的功能，梅林改版也不能使用，比如说UU加速器等；
* 梅林改版固件作为AiMesh节点时可以通过IP地址访问其web后台；
* 固件后台界面优化：字体优化，首页默认显示状态页面；
* 增加一些软件中心/插件需要用到的功能，如dnsmasq with ipset、base64、一些内核模块支持等；
* koolcenter软件中心支持。


#### Asuswrt-Merlin 3006.102.7（2026-02-22）更新日志

- 注意：由于安全记录较差，Asuswrt-Merlin 已移除 AiCloud 功能。

  - Dropbox USB 文件夹同步
