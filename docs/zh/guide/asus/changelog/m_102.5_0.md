**注意事项：**

* 此次刷机后RT-BE88U在澳大利亚模式下的的最大功率会被修改为26dbm，这是华硕官方升级了CFE配置，如果大家用了wifiboost的，需要重新调节下功率。其它机型如果遇到此情况也请自行调整。

* 由于最近华硕官方固件和梅林固件的一些配置调整，建议刷机后做一次双清操作，并且全部手动重新配置路由器设置和软件中心。

* 警告，由于某猫咪图标的第三方插件的兼容性问题，如果你开着此插件升级102.5，可能导致你的路由器开机后卡死，且任何方式无法访问路由器，建议升级102.5之前关闭或者卸载猫咪图标的第三发插件。

------

**刷机注意事项：**

如果你从更高版本的华硕官方固件刷机到梅林固件102.5，遇到固件升级失败，提示映像不符，说明遇到了华硕的防降级机制，关闭固件降级检测方法：ssh登录路由器后台后输入命令：
```
nvram set DOWNGRADE_CHECK_PASS=1
```
输入命令后就能正常使用web刷机的方法进行刷机，以后梅林改刷梅林，梅林改刷官改就不需要此命令。当然，如果以上命令不管用的话，你也可以通过华硕救援模式刷机！


**降级刷机注意事项**

由于梅林改版固件关闭了降级刷机校验，所以如果你进行降级刷机的话，建议刷机后进行一次双清操作。
